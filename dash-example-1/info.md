# Move into folder directory 

```
cd dash-example-1/
```


# To build this image

```
docker build -t dash-example-1 .
```

# To check if image has been created

```
docker images
```


# To run this image

```
docker run -p 8050:8050 dash-example-1 
```


# Access Dashboard 

```
localhost:8050
```
